How does model suggestion work?

Our AI analyzes your prompt and suggests the most suitable model based on the task type, complexity, and required capabilities.

Can I use my own API keys?

Yes, you can use your own API keys for any supported AI model in your account settings.

What's included in the free tier?

The free tier includes 100 prompts per month, basic model suggestions, and access to community features.
